# 📊 Дашборд операторов АЗС

Интерактивный веб-дашборд для анализа эффективности работы операторов на автозаправочных станциях.  
Позволяет отслеживать количество чеков, промо-активность (Coffee & Fuel, Big Races), оплаты по СБП и другие ключевые метрики в разрезе филиалов, АЗС и операторов.

---

## ✨ Возможности

- **📅 Выбор периода** — просмотр данных за произвольный интервал (по умолчанию — текущий месяц)
- **🏢 Фильтр по филиалам и АЗС** — детализация до конкретной станции
- **🏷️ Фильтр по типу акций** — Coffee & Fuel, Big Races или все промо-чеки
- **🏆 Лидеры филиала** — топ-3 оператора по проценту промо-чеков
- **👥 Интерактивный выбор оператора** — кнопки с процентом промо и СБП
- **📈 Стэк-диаграммы** — структура чеков (Promo / Non-Promo) по операторам и филиалам
- **📋 Детальная таблица** — все метрики с подсветкой выбранного оператора
- **📥 Выгрузка в Excel** — экспорт отфильтрованных данных в `.xlsx`

## 🚀 Быстрый старт

### 1. Клонируйте репозиторий

```bash
git clone <url-репозитория>
cd Operators_dashbords_demo
```

### 2. Установите зависимости

```bash
pip install -r requirements.txt
```

### 3. Запустите дашборд

```bash
streamlit run dashboard.py
```

После запуска откроется браузер с дашбордом (по умолчанию `http://localhost:8501`).

---

## 🤖 Автоматическая загрузка данных

Данные подтягиваются из API `https://azs.knp24.ru/api/v1/reports/workers` и сохраняются в `history.csv`.

- **Скрипт:** `fetch_data.py`
- **Логи:** `fetch_data.log`
- **Планировщик:** настроен через **Windows Task Scheduler** на ежедневный запуск в 1:00

Подробная инструкция по настройке — в файле [`TASK_SCHEDULER_SETUP.md`](TASK_SCHEDULER_SETUP.md).

Если данных за сегодня нет, дашборд автоматически загрузит их из API при открытии (фолбэк).
